## 📖 About The Project

**Green Therapy** is a compassionate, word-based emotional healing website designed for Mukulsinhji Chavda (M.Pharm), a pharmacist, academic, and emotional healing guide. The platform offers gentle, non-judgmental healing conversations to help individuals release stress, anxiety, and burnout — without medication.

This is a **demo version** of the website, showcasing the design, layout, and content structure for client review and feedback.

| Service | Description |

## ✨ Features

- ✅ Fully responsive design (mobile, tablet, desktop)
- ✅ Clean, minimalist green theme
- ✅ WhatsApp integration with pre-filled message
- ✅ Contact form for inquiries
- ✅ FAQ section for common questions
- ✅ Step-by-step process explanation
- ✅ Therapist introduction section
- ✅ Quote section with healing message
- ✅ Newsletter subscription
- ✅ AOS animations (Animate on Scroll)
